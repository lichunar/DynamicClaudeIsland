# Dynamic Claude Island 0.1.0

首个可用版本。

## 主要功能

- 可任意拖动位置的悬浮窗，悬停展开，移开收起。
- 自动发现当前 Claude Code 会话。
- 支持切换会话、查看消息、发送消息。
- 消息会尽量发送回原 Claude Code 所在 Terminal 标签页。
- Claude 回复、完成、需要确认、出错时发送通知并改变灵动岛颜色。
- 支持手动关闭并直接退出应用。

## 注意

- 当前发布包未签名。
- 首次运行需要授予通知权限和 Terminal 自动化权限。
- 发送到原 Terminal 会话依赖 Terminal.app 的 AppleScript 支持。
